## 解决方案

**单屏滑动**

    设标准宽高比例值: 1.5

    计算屏幕宽高比例: H / W

    |<-   W   >|
    +----------+  ——
    |          |
    |          |
    |  screen  |  H
    |          |
    |          |
    +----------+  ——

    大于: 以宽度为基准值，得到高度

    |<-   W   >|
    +--------- +  ——
    |----------|
    |          |
    |          |
    |   page   |  H
    |          |
    |          |
    |----------|
    +----------+  ——

    小于:以高度为基准值，得到宽度
    |<-     W     >|
    +--------------+  ——
    | |          | |
    | |          | |
    | |   page   | |  H
    | |          | |
    | |          | |
    +--------------+  ——
